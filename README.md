# WebSiteBuilders

## ファイル構成
|フォルダ名 |説明                         |
|:--        |:--                          |
|top.html       |トップベージ．様々なページにリンクする．|
|event.html     |翔鴎祭2日間の各会場のタイムテーブル．  |
|group.html     |翔鴎祭参加団体の紹介．               |
|access.html    |最寄り駅情報とGoogleMap．           |

## 特記事項

### top.html
以下のものを含む．
* 文化祭の紹介文
* Twitterの埋め込み
* 各ページへのリンク

### event.html
タイムテーブルは広報係から入手する．

### group.html
団体の紹介文とアイコンは広報係から入手する．
